# 多无人机编队控制与协同避障仿真系统 (Multi-UAV Formation & Obstacle Avoidance)

## 📖 项目简介

本项目是一个基于 MATLAB/Simulink 的多旋翼无人机（Quadrotors）三维编队飞行与协同避障仿真平台。项目在底层物理引擎中完美融合了以下两大控制理论：

1. **分布式有限时间一致性协议 (Wang)**：基于领航者-跟随者 (Leader-Follower) 拓扑结构，跟随者仅依赖局部通信网络即可实现对领航者状态的有限时间估计与队形保持。
2. **改进型人工势场法 (Improved APF, Pan)**：解决了传统势场法在障碍物前的“目标不可达 (GNRON)”与“局部极小值死锁”问题。

通过深度的理论适配与工程优化，本系统克服了大型编队在避障时常见的“串级震荡”、“推力气球效应”以及“姿态奇异点”等物理灾难，实现了犹如“水滴绕流”般极其丝滑的柔性编队避障效果。

## ✨ 核心特性 (Features)

* **工业级姿态解算防护**：重写了推力向姿态映射的底层逻辑，加入了防“死亡翻滚 (Death Roll)”的托底保护与基于实际姿态的推力投影，彻底消除了终端微调时的“推力气球效应 (Thrust Ballooning)”。
* **高鲁棒性的柔性编队**：采用动态配平的刚度与阻尼参数 ($k_p, k_v$)，结合优化的前馈切断机制，完美抑制了多智能体网络中的前馈冲击波与“甩鞭效应 (String Instability)”。
* **动态势场叠加与防死锁**：支持在三维空间中自定义“多障碍物/障碍森林”矩阵。引入指数型连续平滑过渡函数，彻底消除了无人机在警戒圈边缘的高频颤振。
* **一键式数据可视化**：提供高度鲁棒的绘图脚本，可自动过滤数值爆炸点，一键生成 3D 静态轨迹图、三轴跟踪误差曲线、避障距离曲线以及极具视觉表现力的 3D 动态飞行过程动画。

## 📂 文件结构 (Project Structure)

* `init_multi.m`：全局初始化脚本。用于配置无人机物理参数、编队拓扑矩阵、期望阵型偏移量 ($\delta$)、控制增益以及三维障碍物矩阵。
* `Multi_Quad.slx` (或主仿真模型)：Simulink 仿真主框架，包含了无人机的非线性动力学模型 (Plant) 与控制信号流。
* `plot_multi.m`：终极仿真结果可视化与动画渲染脚本。

## 🚀 快速开始 (Quick Start)

### 1. 环境要求

* MATLAB / Simulink (建议 R2020a 或更高版本)

### 2. 运行步骤

**步骤一：初始化系统参数**
在 MATLAB 命令行中运行初始化脚本，将物理参数与控制器增益加载到工作区：

```matlab
>> init_multi
```

**步骤二：运行 Simulink 仿真**
打开 Simulink 模型文件，点击 **Run (运行)**。仿真采用定步长（或变步长）求解器，将计算各无人机在三维空间中的非线性动力学响应。

**步骤三：渲染可视化结果**
仿真结束后，直接在命令行运行绘图脚本：

```matlab
>> plot_multi
```

系统将自动弹出 4 个 Figure 窗口，为您全方位展示编队的轨迹、误差收敛情况以及 3D 动态飞行过程。

## ⚙️ 参数调试指南 (Tuning Guide)

如果您想测试不同的编队形态或避障场景，请修改 `init_multi.m` 中的相关参数：

* **修改编队阵型**：调整 `delta` 矩阵（3 x N），每一列代表对应跟随者相对于领航者的期望 XYZ 坐标偏移量。
* **修改障碍物环境**：调整 `obs_center` 矩阵（可无限扩充列数，每列为一个障碍物坐标），并同步调整对应的 `obs_radius`（物理半径）和 `obs_rho0`（警戒半径）数组。
* **控制参数注意**：在当前离散求解器步长下，一致性弹簧位置刚度建议限制在 `kp = 5`，速度阻尼设定在 `kv = 15` 左右，以保持系统处于完美的“微过阻尼”状态，切勿将 $k_p$ 调得过大以免引发数值发散。
